# SSH подключение к серверу

## Задание
Подключиться к VPS от REG.RU по SSH.

## Выполнение
1. Создан сервер в личном кабинете REG.RU
2. Получены данные:
   - IP: `83.166.245.178`
   - Пользователь: `root`
   - Пароль: [из личного кабинета]
3. Подключение через PowerShell:
   ```powershell
   ssh root@83.166.245.178
   ```
4. Принят fingerprint сервера
5. Введен пароль
6. Установлено соединение с Ubuntu 24.04.3 LTS

## Результат
Успешное подключение к серверу по SSH.
